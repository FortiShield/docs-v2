# EgiftcardTransaction Instance Methods

The `EgiftcardTransaction` class provides a set of instance methods to set various properties of your transaction's input data and gas settings.

Do note that in most cases those are autopolulated by the SDK and you don't need to set them manually.

## EgiftcardTransaction.SetTo

Set the target address or contract of the transaction.

```csharp
EgiftcardTransaction transaction = transaction.SetTo("0xToAddress");
```

<Details summary="Parameters">

### toAddress (required)

The target address or contract of the transaction. Must be a hexadecimal `string`.

</Details>

<Details summary="Return Value">

### EgiftcardTransaction

Returns the instance of `EgiftcardTransaction` with the `to` property set.

</Details>

## EgiftcardTransaction.SetData

Set the raw encoded data payload (potentially overriding previous data if the transaction was created using `EgiftcardContract.Prepare`).

```csharp
EgiftcardTransaction transaction = transaction.SetData("0x");
```

<Details summary="Parameters">

### data (required)

The raw data payload of the transaction. Must be a hexadecimal `string`.

</Details>

<Details summary="Return Value">

### EgiftcardTransaction

Returns the instance of `EgiftcardTransaction` with the `data` property set.

</Details>

## EgiftcardTransaction.SetValue

Set the value of the payable transaction in Wei.

```csharp
EgiftcardTransaction transaction = transaction.SetValue(1000000000000000000);
```

<Details summary="Parameters">

### value (required)

The value of the transaction in Wei. Must be a `BigInteger`.

</Details>

<Details summary="Return Value">

### EgiftcardTransaction

Returns the instance of `EgiftcardTransaction` with the `value` property set.

</Details>
