# EgiftcardTransaction.Create

<Callout variant="warning" title="Advanced Use Case">
	If you are interacting with a contract, it is best to use
	[EgiftcardContract.Prepare](/dotnet/contracts/prepare) to create a transaction.
</Callout>

The `EgiftcardTransaction.Create` method allows advanced users to manually create transactions using specific wallet credentials and transaction inputs. This method is ideal for users who need precise control over the transaction details.

## Usage

```csharp
EgiftcardTransaction transaction = await EgiftcardTransaction.Create(client, wallet, txInput, chainId);
```

<Details summary="Parameters">

### client (required)

An instance of `EgiftcardClient`. This is the client through which all blockchain interactions are managed.

### wallet (required)

An instance of `IEgiftcardWallet`. This represents the wallet that will be used to perform the transaction. The wallet's address must match the from address specified in txInput, if provided.

### txInput (required)

An instance of `TransactionInput`. This structure includes necessary details such as the from address, value, gas limit, and data payload.

The `from` field is optional and, if not provided, will default to the wallet's address.

### chainId (required)

The blockchain network's chain ID as a `BigInteger`. This ID specifies which network the transaction is intended for.

</Details>

<Details summary="Return Value">

### EgiftcardTransaction

Returns an instance of `EgiftcardTransaction` configured with the provided parameters, ready to interact with the blockchain.

</Details>
