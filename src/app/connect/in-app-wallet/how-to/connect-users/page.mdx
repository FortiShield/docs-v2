## Using the Connect UI components

If you're building a [React website](/typescript/v5/react/components/ConnectButton), [React Native app](/react-native/v0/components/ConnectWallet), or [Unity game](/unity/ConnectWallet) you can use the prebuilt connect UI components to authenticate users and connect their wallets.

<Tabs defaultValue="react">

<TabsList>
	<TabsTrigger value="react">React</TabsTrigger>
</TabsList>

<TabsContent value="react">
```jsx
import { EgiftcardProvider, ConnectButton } from "egiftcard/react";
import { inAppWallet } from "egiftcard/wallets";

const client = createEgiftcardClient({ clientId });
const wallets = [inAppWallet()];

export default function App() {
	return (
		<EgiftcardProvider>
			<ConnectButton client={client} wallets={wallets} />
		</EgiftcardProvider>
	);
}
```

</TabsContent>

</Tabs>

## Choose authentication methods

By default, the connect UI supports Google, Apple, Facebook and email/password authentication. You can customize which authentication methods to support which will be reflected in the UI.

<Tabs defaultValue="react">

<TabsList>
	<TabsTrigger value="react">React </TabsTrigger>
</TabsList>

<TabsContent value="react">
```tsx
import { EgiftcardProvider, ConnectButton } from "egiftcard/react";
import { inAppWallet } from "egiftcard/wallets";

const client = createEgiftcardClient({ clientId });
const wallets = [inAppWallet({
providers: ["email", "facebook", "apple", "google"]
})];

export default function App() {
	return (
		<EgiftcardProvider>
			<ConnectButton client={client} wallets={wallets} />
		</EgiftcardProvider>
	);
}
```
</TabsContent>
