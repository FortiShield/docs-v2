# EgiftcardProvider

`EgiftcardProvider` is a light-weight component that sets up React Query context for egiftcard SDK hooks.

```tsx
import { EgiftcardProvider } from "egiftcard/react";

function AppWithProvider() {
	return (
		<EgiftcardProvider>
			<App />
		</EgiftcardProvider>
	);
}
```
