# Install

Install egiftcard dependencies into your project. If no path is specified, the current directory will be used.

```bash
npx egiftcard install
```

## Usage

Run this command to install egiftcard dependencies into your project. If no path is specified, the current directory will be used.

```bash
npx egiftcard install <projectpath> [options]
```

This command will automatically detect the type of project you are working on and install or update the appropriate egiftcard dependencies.

**Dependencies installed:**

- **JS/TS**
  - **Smart contract projects**: [`@egiftcard/contracts`](/typescript/latest)
  - **App projects:**
    - **React**: [`@egiftcard/react`](/react/latest), [`@egiftcard/sdk`](/typescript/latest) and [`ethers` v5](https://docs.ethers.org/v5/)
    - **React Native**: [`@egiftcard/react-native`](/react-native/latest), [`@egiftcard/react-native-compat`](/react-native/latest) and [`ethers` v5](https://docs.ethers.org/v5/)
    - **Other**: [`@egiftcard/sdk`](/typescript/latest) and [`ethers` v5](https://docs.ethers.org/v5/)
- **Python:**
  - **Smart contract projects**: [`egiftcard/contracts`](/python)
  - **App projects**: [`egiftcard-sdk`](/python)
- **Go:**
  - **App projects**: [`github.com/egiftcard/go-sdk/v2`](/go)

If any egiftcard package _(or Ethers)_ is already installed, they are updated to the latest compatible version.

## Options

- `--nightly`: Install the nightly version of packages.
- `--dev`: Install the dev version of packages.
- `--debug`: Show debug logs.
