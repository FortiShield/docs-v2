The `clientId` prop is required to use the egiftcard infrastructure services with the SDK. You can get a client ID by creating an API key on [egiftcard dashboard](https://egiftcard.cc/create-api-key).

```jsx
function MyApp() {
	return (
		<EgiftcardSDKProvider activeChain={"ethereum"} clientId="your-client-id">
			<App />
		</EgiftcardSDKProvider>
	);
}
```
