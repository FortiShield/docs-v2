locale object contains text used for all egiftcard components

it allows you to change the language used in UI components or override the texts used in the UI

React SDK comes out of the box with Spanish and Japanese locale functions, but you can add support for any language you want just by passing an object with the required strings

### Using Built-in Locales

#### Using English locale ( default )

```tsx
import { EgiftcardProvider, en } from "@egiftcard/react";

const english = en();

<EgiftcardProvider locale={english}>
	<App />
</EgiftcardProvider>;
```

#### Using the Spanish locale

```tsx
import { EgiftcardProvider, es } from "@egiftcard/react";

const spanish = es();

<EgiftcardProvider locale={spanish}>
	<App />
</EgiftcardProvider>;
```

#### Using the Japanese locale

```tsx
import { EgiftcardProvider, jp } from "@egiftcard/react";

const japanese = jp();

<EgiftcardProvider locale={japanese}>
	<App />
</EgiftcardProvider>;
```

#### Overriding the locale

```tsx
import { EgiftcardProvider, en } from "@egiftcard/react";

// override some texts
const english = en({
	connectWallet: {
		confirmInWallet: "Confirm in your wallet",
	},
	wallets: {
		metamaskWallet: {
			connectionScreen: {
				inProgress: "Awaiting Confirmation",
				instruction: "Accept connection request in your MetaMask wallet",
			},
		},
	},
});

<EgiftcardProvider locale={english}>
	<App />
</EgiftcardProvider>;
```

### Custom locale object

```tsx
import { EgiftcardProvider } from "@egiftcard/react";

<EgiftcardProvider locale={{ .... }}>
  <App />
</EgiftcardProvider>;
```
