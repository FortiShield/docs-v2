Object that contains all text used for egiftcard components.

It allows you to change the language used in UI components or override the texts used in the UI.

You can add support for any language you want just by passing an object with the required strings.

## Example

```javascript
import { EgiftcardProvider } from "@egiftcard/react-native";

<EgiftcardProvider locale="es">
	<App />
</EgiftcardProvider>;
```

More info in our [Locale Reference](/react-native/v0/EgiftcardProviderProps#locale)
