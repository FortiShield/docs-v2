Sets the theme for all egiftcard components.

By default it is set to "dark".

theme can be set to either "dark" or "light" or a custom theme object.

You can also import the `lightTheme` or `darkTheme` functions from `@egiftcard/react-native` to use the default themes as base and overrides parts of it.

## Example

```javascript
import { EgiftcardProvider } from "@egiftcard/react-native";

<EgiftcardProvider theme="light">
	<App />
</EgiftcardProvider>;
```

More info in our [Theme Reference](/react-native/v0/EgiftcardProviderProps#theme)
