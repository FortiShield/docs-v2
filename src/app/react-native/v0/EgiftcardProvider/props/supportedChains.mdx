An array of chains supported by your app.

You can import chains from `@egiftcard/chains` which contains 1000+ chains.

```jsx
import { EgiftcardProvider } from "@egiftcard/react-native";
import { Ethereum, Polygon } from "@egiftcard/chains";

function MyApp() {
	return (
		<EgiftcardProvider
			activeChain={Ethereum}
			supportedChains={[Ethereum, Polygon]}
		>
			<App />
		</EgiftcardProvider>
	);
}
```

If not provided, It defaults to below shown default chains.
